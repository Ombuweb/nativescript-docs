[#iterations]
=== iterations

• `Optional` *iterations*: `number`

Specifies how many times the animation should be played.
Default is 1.
iOS animations support fractional iterations, i.e.
1.5.
To repeat an animation infinitely, use Number.POSITIVE_INFINITY
